Software

O software utilizado para a impressão 3D é o Marlin Firmware, o software de controle do firmware para impressoras 3D. Ele é responsável por gerenciar as operações da impressora e executar os comandos recebidos do software de fatiamento. Também é reponsável por fornecer uma interface para configurar as opções de impressão. Este mesmo Firmware que é usado, por exmeplo, nos primeiros modelos da impressora Creality Ender 3.
