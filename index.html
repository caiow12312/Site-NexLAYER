SOFTWARE

Em nosso projeto utilizamos o Marlin Firmaware (com nossas adaptações e correções), que é a linguagem que comanda a impressora(no caso a linguagem de programação C++), sendo feita para o micro controlador Arduino MEGA 2560. Para modelar peças, recomendamos o Autodesk Fusion 360, porém você caso não tenha você pode pegar peças já prontas para imprimir em sites(Make Word, por exemplo), ou usar ferramentas gratuitas para modelagem como Blender, ThinkerCad, entre outros. Já para fatiar, recomendamos o OrcaSlicer, um fatiador universal completo e constatemente atualizado.
